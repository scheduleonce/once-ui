# SlideToggle

`<oui-slide-toggle>` is an on/off control that can be toggled via clicking. The slide-toggle behaves similarly to a checkbox. A `<oui-slide-toggle>` element should be used whenever some toggle action is performed. See: https://github.com/scheduleonce/once-ui/blob/master/ui/src/components/slide-toggle/README.md
