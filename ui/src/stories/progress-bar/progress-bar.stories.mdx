# ProgressBar

`<oui-progress-bar>` is a horizontal progress-bar for indicating progress and activity.

Example:

Determinate: `<oui-progress-bar color="primary|accent|warn"></oui-progress-bar>`

Indeterminate: `<oui-progress-bar color="primary|accent|warn" value="1 to 100"></oui-progress-bar>`

See: https://github.com/scheduleonce/once-ui/blob/master/ui/src/components/progress-bar/README.md
