# Icon

OuiIcon component to display an icon. It can be used in the following ways:

Specify the svgIcon input to load an SVG icon from a URL previously registered with the addSvgIcon, addSvgIconSet. Examples: `<oui-icon svgIcon="[name]"></oui-icon> <oui-icon svgIcon="building"></oui-icon>`. See https://github.com/scheduleonce/once-ui/blob/master/ui/src/components/icon/README.md

Icon with Default color:

<Canvas>
  <Story of={IconStories.DefaultColor} />
</Canvas>

You can change size and use specific colors on `Icon` component by using css:

<Canvas>
  <Story of={IconStories.CustomIcon} />
</Canvas>
