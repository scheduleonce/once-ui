# Icon

OuiIcon component to display an icon. It can be used in the following ways:

Specify the svgIcon input to load an SVG icon from a URL previously registered with the addSvgIcon, addSvgIconSet. Examples: `<oui-icon svgIcon="[name]"></oui-icon> <oui-icon svgIcon="building"></oui-icon>`. See https://github.com/scheduleonce/once-ui/blob/master/ui/src/components/icon/README.md

Icon with Default color:

<Canvas>
  <Story
    name="Default color"
    parameters={{
      docs: {
        source: {
          code: `<oui-icon svgIcon="calendar" color="primary"></oui-icon>`,
        },
      },
    }}
  >
    {{
      moduleMetadata: {
        imports: [OuiIconModule],
        schemas: [],
        declarations: [OuiiconStorybook],
      },
      template: `<oui-icon-storybook 
                [icon]="icon" 
                [size]="size">
                </oui-icon-storybook>`,
      props: {
        icon: text('icon', 'calendar'),
      },
    }}
  </Story>
</Canvas>

You can change size and use specific colors on `Icon` component by using css:

<Canvas>
  <Story
    name="Custom icon"
    parameters={{
      docs: {
        source: {
          code: `<oui-icon svgIcon="configuration" class="maroon"></oui-icon>`,
        },
      },
    }}
  >
    {{
      moduleMetadata: {
        imports: [OuiIconModule],
        schemas: [],
        declarations: [OuiiconStorybook],
      },
      template: `<oui-icon-storybook 
                [color]="color" 
                [icon]="icon" 
                [size]="size">
                </oui-icon-storybook>`,
      props: {
        color: select('color', ICON_COLORS, ICON_COLORS[0]),
        icon: text('icon', 'configuration'),
        size: number('size', 20),
      },
    }}
  </Story>
</Canvas>
