# Scrollbar

The `oui-scrollbar` provides a custom scrollbar to the inner content of element it has been applied on.

All radio-buttons with the same name comprise a set from which only one may be selected at a time. See https://github.com/scheduleonce/once-ui/blob/master/ui/src/components/radio/README.md
