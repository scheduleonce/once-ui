# ProgressSpinner

`<oui-progress-spinner>` is a circular indicators of progress and activity.

Example:

Determinate: `<oui-progress-spinner color="primary|accent|warn"></oui-progress-spinner>`

Indeterminate: `<oui-progress-spinner color="primary|accent|warn" value="1 to 100"></oui-progress-spinner>`

See: https://github.com/scheduleonce/once-ui/blob/master/ui/src/components/progress-spinner/README.md
